# MiddleGlyph Font Rendering

MiddleGlyph is a square bitmap font designed for embedded displays such as the SSD1306 OLED.

This project provides a custom 12x12 glyph set along with rendering routines optimized for direct framebuffer manipulation on resource-constrained systems like the ESP32.

## Features

- 12x12 square bitmap font
- Optimized for SSD1306 displays
- Pixel-level rendering
- Smooth horizontal scrolling
- Scalable glyph rendering (12x12 → 24x24)
